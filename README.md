# Gaming Website

A fully responsive gaming landing page with cinematic visuals and premium UI styling.

## Files

- `index.html` — landing page markup
- `styles.css` — responsive cinematic styling
- `script.js` — mobile navigation and contact form behavior

## Usage

1. Open `index.html` in a browser to preview.
2. Replace the placeholder text and images with your own game titles and copy.
3. Customize the section titles, event names, and contact details to match your brand.

## Notes

This is a static website for showcasing gaming events, featured titles, and community engagement. It can be hosted on GitHub Pages or any static hosting platform.
